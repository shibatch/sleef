Data types for AArch64 architecture

Sleef_float32x4_t_2

Description

Sleef_float32x4_t_2 is a data type for storing two float32x4_t values, which is defined in sleef.h as follows: 

typedef struct {
  float32x4_t x, y;
} Sleef_float32x4_t_2;
